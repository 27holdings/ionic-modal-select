Callbacks

Selection callback

You can set a callback that will be fired when the user selects one option with the on-select attribute. This attributes accepts a function call with the newValue and oldValue arguments. You must use these exact argument names in your template, otherwise the callback won't work.

Controller:

$scope.selectables = [1, 2, 3]; $scope.shoutLoud = function(newValue, oldValue){ alert("changed from " + JSON.stringify(oldValue) + " to " + JSON.stringify(newValuea)); };

Template

Selected value: {{onSelectModel}}

Result:

Selected value: {{onSelectModel}}

Reset callback

You can set a callback that will be fired when the user resets the selected option one option with the on-reset attribute. This attributes accepts a function call with no arguments.

Controller:

$scope.selectables = [1, 2, 3]; $scope.shoutReset = function(){ alert("value was reset!"); };

Template

Selected value: {{onResetModel}}

Result:

Selected value: {{onResetModel}}
