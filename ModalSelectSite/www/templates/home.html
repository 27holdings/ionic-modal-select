ionic-modal-select

A modal select ui component for the Ionic framework

An ng-model aware directive that turns any clickable element into a select widget that shows selectable options in a modal.

Docs
Examples
Features

Long lists

Supports long list of object via collection-repeat

Customizable UI

Buttons and modal style

Callbacks

Callbacks for item selection and selection reset

Search bar

Optional search bar to filter options.
