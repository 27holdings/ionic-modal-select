Array of objects

Getting whole object - Multiple mode

The same options getters for the single selection work with multiple mode. You can pass in an array of objects to be used as options. In this case you can decide which fields you want to render in your option template. If nothing else is specified, the directive will set the whole object on your model

Controller:

$scope.secondModelMultiple = []; $scope.selectableNames = [ { name : "Mauro", role : "juggler"}, { name : "Silvia", role : "chef"}, { name : "Merlino", role : "little canaglia"}, ];

Template:

Selected value: {{secondModelMultiple}}

Result:

Selected value: {{secondModelMultiple}}

Getting a single property - Multiple selection

If you want to pick only a property of your object as result of selection, you can do it by using the option-property attribute.
In our previous example, let's pick only the value of the "name" property of our options. Note that you can still use all the properties of your objects in option template.

Template:

Selected value: {{model3Multiple}}

Result:

Selected value: {{model3Multiple}}
