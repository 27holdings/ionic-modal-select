Array of values - Multiple selections

You can turn on the multiple selction by setting the multiple="true" attribute on the directive element. This will allow the user to select more options by means of checkboxes.

Controller:

$scope.someModelMultiple = []; $scope.selectables = [1, 2, 3];

Template:

someModelMultiple value: {{someModelMultiple}}

Result:

someModelMultiple value: {{someModelMultiple}}
