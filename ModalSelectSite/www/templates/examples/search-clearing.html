Clearing search bar after selection

When you add the search bar and the user selects an option, an eventual search text will be cleared after the selection. If you want to persist the search text after the selection, set clear-search-on-select to false.

Controller:

$scope.numbers = [1,2,3,4,5,6,7,8];

Template:

Selected value: {{persistSearchModel}}

Result:

Selected value: {{persistSearchModel}}
