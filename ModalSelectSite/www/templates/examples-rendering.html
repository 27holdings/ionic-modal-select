Rendering

Automatic rendering mode

modal-select directive has two rendering modes: one based on ng-repeat that renders all the items and a smarter one based on ionic collection-repeat that is useful if you need to render long lists of options.

The directive chooses automatically a rendering mode based on the number of available options. When the nummber of options is equal or greater than the value set by the short-list-break attribute (which defaults to 10) collection-repeat will be used.

In this example we show two selects on the same options and model, with two different breakpoints: 10 (default) and 5

Controller:

$scope.numbers = [1,2,3,4,5,6,7,8];

Template:

Selected value: {{autoRenderingModel}}

Result:

Selected value: {{autoRenderingModel}}

Explicit rendering mode

Rendering mode can be selected explicitly with the use-collection-repeat attribute

Here's an example of two modal selects based on the same options with a different setting of this attribute.

Controller:

$scope.numbers = [1,2,3,4,5,6,7,8];

Template:

Selected value: {{useCollRepeatModel}}

Result:

Selected value: {{useCollRepeatModel}}

Setting the loading message

When using collection-repeat rendering mode, you can set the message on the loader with the load-list-message attribute.

Controller:

$scope.numbers = [1,2,3,4,5,6,7,8];

Selected value: {{loadListModel}}
